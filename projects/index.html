Basic template

All HTML documents build on the following basic formatting:

            
<!DOCTYPE html>
<html>
    <head>
    
    </head>

    <body>

    </body>
</html>

  • <!DOCTYPE html> tells the browser this is a HTML5 document.

  • <html> contains all HTML code.

  • <head> contains the metadata of a page: its title, links to stylesheets, etc.

  • <body> contains the visible content of a page.

Tags

Some basic ones

Tag Use
<title> Page title (placed within <head>) that displays in the tab of modern browsers.
<p> Paragraph
<h1> to <h6> Headings
<img> Image; source set with href attribute
<video> Video; source set with href attribute
<em> Emphasises text (italicises by default)
<strong> Stronger emphasis (bolds by default)
<a> Hyperlink; target set with href attribute
<div> Nonspecific division tag. Useful for styling when combined with class and id attributes.

Semantic tags

These make a page's HTML code much more readable than <div> soup. They also make the page more accessible for screen reader users.

Examples I've been using:

Tag Use
<nav> Navigation menu
<section> Groups together related elements with a heading
<aside> An aside from the main text
<figure> Contains a figure and its caption
<figcaption> Figure caption

Styling

Ways to style a page

  • Within individual tags

    CSS declarations can be placed within HTML tags using the style attribute. For example, <p style="color: red"> will turn a paragraph red.

    This is inadvisable; you have to configure styles for every individual element on every page.

  • Within a <style> element

    CSS code can be placed within a <style> element within a HTML document's <head> element.

    This is much better than the previous method, but it still isn't best practice as every page of a site must be styled individually.

  • Using a stylesheet

    A .css file containing CSS code can be used to style a webpage by placing this within the HTML document's <head> element:

    <link href="[path to file]" rel="stylesheet">

    This is best practice; the same stylesheet can be used to style infinite pages.
